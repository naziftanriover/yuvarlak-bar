⭐ Legend of the West — Otomatik Açılış (2 ekran)

Bu sefer kasa penceresi otomatik olarak ana ekrana taşınacak. İki dosya oluşturacağız; ikisi de aynı klasörde (Başlangıç) olacak.

Önce eski dosyayı sil: Başlangıç klasöründe daha önce koyduğun LegendWest.bat varsa onu sil, yeniden yapacağız.

DOSYA 1 — LegendWest.ps1


  
  
    
    
  1. Notepad aç → yapıştır.
    2. 
    
  2. File → Save As → "Save as type" = All Files.
    3. 
    
  3. Dosya adı: LegendWest.ps1 → Başlangıç klasörüne kaydet.
    4. 
  


  
DOSYA 2 — LegendWest.bat

  
@echo off
powershell -NoProfile -ExecutionPolicy Bypass -File "%~dp0LegendWest.ps1"

  
  
    
    
  1. Yeni bir Notepad aç → yapıştır.
    2. 
    
  2. File → Save As → "Save as type" = All Files.
    3. 
    
  3. Dosya adı: LegendWest.bat → aynı Başlangıç klasörüne kaydet.
    4. 
  


  
Başlangıç klasörünü açmak: Dosya Gezgini'nin adres çubuğuna shell:startup yazıp → oka bas. İki dosya da bu klasörde olmalı.


  
Test et

  
    
    
  1. LegendWest.bat'a çift tıkla.
    2. 
    
  2. ~10 saniye bekle: arka ekranda müşteri ekranı, sonra kasa penceresi açılıp kendiliğinden ana ekrana geçecek.
    3. 
    
  3. Olduysa 🎉 bilgisayarı yeniden başlat — açılışta otomatik gelir.
    4. 
  


  
Not: Bilgisayar açıkken test ederken mouse'a/klavyeye dokunma, bırak kendi yapsın (pencereyi kendisi taşıyacak). Yine olmazsa dert etme — bana yaz, en basit yönteme geçeriz (sabah tek dokunuş).
