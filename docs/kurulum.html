⭐ Legend of the West — Otomatik Açılış Kurulumu

Bu sayfayı barın bilgisayarında (BIISON) açtın demektir. Aşağıdaki adımlarla, bilgisayar her açıldığında iki ekran kendiliğinden hazır gelecek.

1) Aşağıdaki metni kopyala

@echo off
set U=https://naziftanriover.github.io/yuvarlak-bar
start "" chrome --kiosk-printing --no-first-run --disable-session-crashed-bubble --window-position=1366,0 --window-size=1366,768 --app=%U%/musteri.html
timeout /t 4 >nul
start "" chrome --new-window %U%/giris.html

2) Notepad'e yapıştır ve kaydet

  1. Başlat → Notepad aç.
  2. Kopyaladığın metni içine yapıştır (Ctrl+V).
  3. Menüden File → Save As.
  4. Alttaki "Save as type" kısmını "All Files" yap (çok önemli — yoksa .txt olur).
  5. Dosya adını LegendWest.bat yaz.
  6. Kaydedilecek yer olarak Başlangıç (Startup) klasörünü seç (bir sonraki adım). Kaydet.

3) Başlangıç klasörünü aç (kaydederken buraya kaydet)

  1. Klavyede Windows tuşu + R.
  2. Açılan kutuya shell:startup yaz → Enter.
  3. Açılan klasör Başlangıç klasörü. Notepad'de "Save As" yaparken bu klasörü seçip LegendWest.bat olarak kaydet.

4) Test et

  1. Başlangıç klasöründeki LegendWest.bat dosyasına çift tıkla.
  2. Chrome kapanıp yeniden açılmalı: arka ekranda müşteri ekranı, ön ekranda kasa.
  3. Çalıştıysa 🎉 bilgisayarı bir yeniden başlat — açılışta kendiliğinden gelecek.

Not: Kasa penceresi yanlış ekrana gelirse bir kez ana ekrana sürükle. "Windows protected your PC" çıkarsa → More info → Run anyway. Sorun olursa ekranın fotoğrafını at.
