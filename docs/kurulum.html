⭐ Legend of the West — Otomatik Açılış (canlı hesaplı)

Kasa penceresi otomatik ana ekrana taşınır, müşteri ekranı arkada canlı hesabı gösterir. İki dosya — ikisi de aynı klasörde (Başlangıç).

Önce eski dosyaları sil: Başlangıç klasöründe eski LegendWest.bat / LegendWest.ps1 varsa hepsini sil.

DOSYA 1 — LegendWest.ps1

yükleniyor…
  1. Notepad aç → yapıştır.
  2. File → Save As → "Save as type" = All Files → adı LegendWest.ps1 → Başlangıç klasörüne kaydet.

DOSYA 2 — LegendWest.bat

yükleniyor…
  1. Yeni Notepad → yapıştır.
  2. File → Save As → "Save as type" = All Files → adı LegendWest.bat → aynı Başlangıç klasörüne kaydet.
Başlangıç klasörü: Dosya Gezgini adres çubuğuna shell:startup yazıp aç. İki dosya da bu klasörde olmalı.

Test

  1. LegendWest.bat'a çift tıkla.
  2. ~15 saniye hiçbir şeye dokunmadan bekle. Müşteri ekranı arkada açılır, kasa penceresi açılıp kendiliğinden ana ekrana geçip büyür.
  3. Doğruysa 🎉 bilgisayarı yeniden başlat.

Test sırasında mouse/klavyeye dokunma, bırak kendi taşısın. "Windows protected your PC" → More info → Run anyway. Yine olmazsa yaz, en basit (marka ekranlı) sürüme döneriz.
