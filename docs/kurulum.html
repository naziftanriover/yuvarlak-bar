⭐ Legend of the West — Otomatik Açılış (kesin sürüm)

Bu sürüm iki ekranı da doğrudan kendi yerine açar — tek dosya, taşıma yok.

Önce eski dosyaları sil: Başlangıç klasöründe daha önce koyduğun LegendWest.bat ve LegendWest.ps1 varsa ikisini de sil.

1) Aşağıdaki metni kopyala


   

  
2) Notepad'e yapıştır ve kaydet

  
    
    
  1. Notepad aç → yapıştır (Ctrl+V).
    2. 
    
  2. File → Save As → alttaki "Save as type"ı All Files yap.
    3. 
    
  3. Dosya adı: LegendWest.bat
    4. 
    
  4. Kaydedilecek yer: Başlangıç klasörü — Dosya Gezgini adres çubuğuna shell:startup yazıp aç, oraya kaydet.
    5. 
  


  
3) Test et

  
    
    
  1. LegendWest.bat'a çift tıkla.
    2. 
    
  2. Açık Chrome'lar kapanır, sonra: arka ekranda müşteri (karşılama) ekranı, ön ekranda kasa açılır.
    3. 
    
  3. Doğruysa 🎉 bilgisayarı yeniden başlat — açılışta otomatik gelir.
    4. 
  


  
Not: Bu sürümde müşteri ekranı marka/karşılama gösterir. Ekranlar yine ters açılırsa (çok düşük ihtimal) bana yaz. "Windows protected your PC" çıkarsa → More info → Run anyway.
